Difference between padding,margin and border

Padding:This is used to set the border on an element. The CSS border properties (border-style, border-color, border-width and border-radius) are used to specify the style, color and size of the border of an element.

Margin:The margin property is used to define the space around elements. It clears an area around the element. Let's get a clear picture of this by actually coding it.

Border:is used to set the border on an element. The CSS border properties (border-style, border-color, border-width and border-radius) are used to specify the style, color and size of the border of an element.
